# EasyVisa Data Analysis Project

This repository contains the full workflow for the EasyVisa data analysis and modeling project.  
It includes exploratory data analysis (EDA), preprocessing, feature engineering, and model development.

##  Project Contents
- **EasyVisa_Notebook.ipynb** — Full Python notebook with code, analysis, and results  
- **EasyVisa_Notebook.html** — Exported HTML version for easy viewing  


##  Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Jupyter Notebook
- Matplotlib & Seaborn

##  How to Use
1. Clone or download the repository: git clone https://github.com/maurynbotha/easyvisa-project.git

2. 2. Open the notebook:
jupyter notebook Project_Full_Code_Notebook_EasyVisa.ipynb


##  Overview
This project covers:
- Data cleaning & preprocessing
- Exploratory data analysis (EDA)
- Feature engineering
- Building and evaluating machine learning models
- Insights and recommendations
